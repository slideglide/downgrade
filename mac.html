macOS Guide for Downgrading to 2.2074

You must own Geometry Dash on Steam for this guide to work.
It is recommended to save your account data to the cloud before proceeding with this guide.

Note: This guide assumes Geometry Dash is installed to the default Steam install location. If Geometry Dash is stored somewhere else, the paths you need to use will be different.

Click here to open the Steam console, or enter steam://open/console in the address bar of Safari or another browser.

Paste the following command into the console and press Enter:

  • download_depot 322170 322172 8771064225227566581

Wait for the download to finish.

Open Finder and press ⌘ Cmd + Shift + . to enable viewing hidden files if you haven’t already.

Copy Geometry Dash.app from the following location:

  • {your user folder}/Library/Application Support/Steam/Steam.AppBundle/Steam.app/Contents/MacOS/steamapps/content/app_322170/depot_322172/Geometry Dash.app
Tip: Right-click Steam.app and choose View Contents to access this folder.

Paste the app into the directory below and replace the existing file if prompted:

  • {your user folder}/Library/Application Support/Steam/steamapps/common/Geometry Dash

To prevent Steam from automatically updating Geometry Dash, go to:

  • {your user folder}/Library/Application Support/Steam/steamapps

Right-click appmanifest_322170.acf, select Get Info, and enable the Locked checkbox.

After Downgrading

You can now launch Geometry Dash through Steam and optionally install Geode v4.10.1 from the Geode website.

If Geometry Dash receives a new update, you will need to launch it through Geometry Dash.app in the game's folder, or temporarily remove Locked from appmanifest_322170.acf to update Geometry Dash then follow this guide to downgrade it again.
