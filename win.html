Windows Guide for Downgrading to 2.2074

Open any web browser (such as Chrome or Edge) and enter steam://open/console in the address bar to open the Steam console.

Paste the following command into the console and press Enter:

  • download_depot 322170 322171 7678373534998244044

Wait for Steam to finish downloading the depot.

Open File Explorer and navigate to the following folder:

  • C:\Program Files (x86)\Steam\steamapps\content\app_322170\depot_322171

Copy everything inside this folder.

Go to the Geometry Dash installation directory:

  • C:\Program Files (x86)\Steam\steamapps\common\Geometry Dash

Paste the files and replace all existing files if prompted.

To prevent Steam from automatically updating Geometry Dash, go to:

  • C:\Program Files (x86)\Steam\steamapps

Right-click appmanifest_322170.acf, select Properties, enable the Read-only checkbox, click Apply, then press OK.

After Downgrading

You can now launch Geometry Dash through Steam and optionally install Geode v4.10.1 from the Geode website.
