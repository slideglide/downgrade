Linux Guide for Downgrading to 2.2074

You must own Geometry Dash on Steam for this guide to work.
It is recommended to save your account data to the cloud before proceeding with this guide.

Note: This guide assumes Geometry Dash is installed to the default Steam install location. If Geometry Dash is stored somewhere else, the paths you need to use will be different.

Click here to open the Steam console, or type xdg-open steam://open/console in your terminal.

Paste the following command into the console and press Enter:

  • download_depot 322170 322171 7678373534998244044

Wait for Steam to finish downloading the depot.

Open and navigate to the following folder:

  • ~/.local/share/Steam/steamapps/content/app_322170/depot_322171

Copy everything inside this folder.

Go to the Geometry Dash installation directory:

  • ~/.local/share/Steam/steamapps/common/Geometry Dash

Paste the files and replace all existing files if prompted.

To prevent Steam from automatically updating Geometry Dash, type this in your terminal to make the manifest file read only:

  • chmod 444 ~/.local/share/Steam/steamapps/appmanifest_322170.acf

After Downgrading

You can now launch Geometry Dash through Steam and optionally install Geode v4.10.1 from the Geode website.

If Geometry Dash receives a new update, you will need to launch it through GeometryDash.exe in the game's folder, or run chmod 644 ~/.local/share/Steam/steamapps/appmanifest_322170.acf to temporarily make the file writeable to update Geometry Dash, then follow this guide to downgrade it again.
